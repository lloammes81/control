📋 Como obtener la Anon Key

  1. Ve a supabase.com y selecciona tu proyecto
  2. Menu izquierdo → Settings → API
  3. Copia la clave anon · public (empieza con eyJ)
  4. Pega arriba y presiona Guardar luego Probar conexion

🗄 SQL para crear las 4 tablas necesarias

  1. Ve a SQL Editor en tu dashboard de Supabase
  2. Crea: bills, payments, credit_accounts, credit_payments, user_profile
  3. Deshabilita RLS: ALTER TABLE nombre DISABLE ROW LEVEL SECURITY;
  4. SQL para user_profile:
    CREATE TABLE user_profile (id TEXT PRIMARY KEY DEFAULT 'main', name TEXT, last_name TEXT, email TEXT, phone TEXT, dob TEXT, address TEXT, city TEXT, state TEXT, notes TEXT, photo TEXT, pin_hash TEXT, pin_enabled BOOLEAN DEFAULT false, updated_at TIMESTAMPTZ DEFAULT now()); ALTER TABLE user_profile DISABLE ROW LEVEL SECURITY;
